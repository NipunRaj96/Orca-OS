## ✅ Best Working Commands for Presentation

### 1. System Monitoring Commands

```bash
# Disk Usage
python3 -m orca.cli "show me disk usage"
# Shows: Complete disk usage with all partitions

# Memory Usage  
python3 -m orca.cli "check memory usage"
# Shows: Detailed memory statistics

# System Information
python3 -m orca.cli "show me system information"
# Shows: System kernel and version info

# System Health Score
python3 -m orca.cli "show health score"
# Shows: Comprehensive health score with recommendations
```

### 2. File Operations

```bash
# List Current Directory
python3 -m orca.cli "list all files in the current directory"
# Shows: All files including hidden ones

# Organize Files
python3 -m orca.cli "organize my downloads"
# Organizes: Files into categories automatically

# Current Working Directory
python3 -m orca.cli "where am I"
# Shows: Current directory path
```

### 3. System Optimization

```bash
# Optimize System
python3 -m orca.cli "optimize my system"
# Automatically: Detects and fixes issues

# System is Slow
python3 -m orca.cli "my system is slow"
# Analyzes: Performance bottlenecks and fixes them
```

### 4. Analytics & Insights

```bash
# Usage Patterns
python3 -m orca.cli "show my usage patterns"
# Shows: Productivity metrics and insights

# Analytics Dashboard
python3 -m orca.cli "analytics"
# Shows: Complete analytics dashboard
```

### 5. Interactive Mode

```bash
# Start Interactive Chat
python3 -m orca.cli --interactive

# Then try:
You: show me disk usage
You: organize my downloads
You: show health score
You: exit
```

### 6. History & Favorites

   ```bash
# View History
python3 -m orca.cli --history

# Search History
python3 -m orca.cli --history-search "organize"

# View Favorites
python3 -m orca.cli --favorites

# View Templates
python3 -m orca.cli --templates
```
